# ResearchAgent

Researches any topic and compiles findings into a structured markdown report.

## Installation

```bash
pip install -r requirements.txt
cp .env.example .env  # add your ANTHROPIC_API_KEY
```

## Usage

```bash
python research_agent.py "your topic here"
```

## Model

`claude-sonnet-4-20250514` via the Anthropic Python SDK.
